# Idea 8 — Gasless Treasury Simulator on Status

## One-liner
A testnet deployment of the treasury + delegation primitive on Status Network Sepolia, demonstrating gasless creation, policy updates, and safe spend attempts.

## Why it looks strong
Mostly a **cheap bonus path**. If we already build a contract-heavy core, this is an easy additional badge.

## Core primitive
- Deploy treasury primitive on Status testnet
- Execute one or more gasless transactions
- Show the agent creating or updating budget policy without fee friction

## Best target tracks
- Go Gasless: Deploy & Transact on Status Network with Your AI Agent
- Any contract-centric main track we already target

## Novelty
Low on its own, but useful as a low-cost add-on.

## Risk

## Build scope
Low.
